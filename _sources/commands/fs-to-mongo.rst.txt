fs-to-mongo
===========

.. code-block:: bash

	usage: regolith fs-to-mongo [-h] [--host HOST] [--uri URI]

	optional arguments:
	  -h, --help   show this help message and exit
	  --host HOST  Specifies a resolvable hostname for the mongod to which to
	               connect. By default, the mongoimport attempts to connect to a
	               MongoDB instance running on the localhost on port number 27017.
	  --uri URI    Specify a resolvable URI connection string (enclose in quotes)
	               to connect to the MongoDB deployment.
